Exercicio - Lista de Compras

Lista de Compras Desenvolva um sistema de lista de compras com as funcionalidades: Adicionar item (nome e quantidade) Listar todos os itens Remover item por nome Marcar item como comprado Exibir apenas itens não comprados Limpar lista completa Sair Cada item deve ser um objeto com: nome, quantidade e status (comprado/não comprado).
